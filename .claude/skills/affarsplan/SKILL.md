# Affärsplan: Menysynk API

All research nedan är gjord åt dig av Nexar Academy. Använd den när du planerar MVP:n och tar affärsbeslut.

## 🎯 Möjligheten

Ett API som synkroniserar restaurangmenyer i realtid mellan kassasystem, bokningsplattformar, leveransappar och digitala menyer. Restauranger uppdaterar menyn på ett ställe och förändringen sprids automatiskt till alla anslutna kanaler. Minskar manuellt dubbelarbete och felaktiga menyer på tredjepartsplattformar.

| Datapunkt | Värde |
|---|---|
| Sökvolym (Sverige) | okänt |
| CPC (annonspris) | okänt |
| Konkurrensnivå | Medel |
| Prisspann i marknaden | 299–1 499 SEK |
| Uppskattad intäkt | 15 000–120 000 kr/mån |
| Svårighet | Svår |
| Tid till första intäkt | 3–6 månader |
| Möjlighets-score | 6/10 |

**Varför nu:** Restaurangbranschen hanterar allt fler parallella försäljningskanaler (Wolt, Foodora, Google, egna appar) och kravet på korrekt allergeninformation ökar med EU-lagstiftning. Manuell menyhantering per plattform är ohållbar och skapar fel som skadar förtroendet. Restaurang-AI-verktyg exploderar 2026 och alla behöver strukturerad menydata som grund.

## 🥊 Konkurrenter & deras svaghet

- **Resos** (https://resos.com/sv/restaurant-api/), svaghet: Fokuserar på bokning och takeaway-integrationer, inte dedikerad menysynkronisering i realtid mot flera kanaler.
- **Syncky** (https://syncky.com/sv/), svaghet: Erbjuder digital meny och QR-kod men saknar ett öppet API för djupare systemintegrationer.
- **GloriaFood** (https://www.gloriafood.com/sv/bestallningssystem-for-restauranger-med-api-for-matbestallning), svaghet: API täcker orderprocessning och meny för det egna systemet, men är inte byggt som en neutral synk-hub mot externa plattformar.

## 💡 Erbjudandet (förslag, justera fritt)

- **Produkt:** Menysynk API, Uppdatera menyn en gång. Alla kanaler synkas automatiskt, direkt.
- **Pitch:** Menysynk API är den neutrala synk-hubben som kopplar ihop ditt kassasystem med Wolt, Foodora, Google, bokningsplattformar och digitala menyer i realtid. Du gör en ändring och den sprids automatiskt till varje ansluten kanal inom sekunder. Inga mer felaktiga priser på leveransappar, ingen manuell dubbelinmatning och full kontroll över allergeninformation enligt EU-krav.
- **Pris:** 699 SEK/mån
- **Målgrupp:** Svenska restauranger, kedjor och restauranggrupper som säljer via flera parallella kanaler som Wolt, Foodora och egna appar, och som idag hanterar menyuppdateringar manuellt per plattform.
- **Unika säljpunkter:**
1. Neutral synk-hub: fungerar mot alla externa plattformar, inte bara ett eget slutet system
2. Realtidssynkronisering av menyändringar till samtliga kanaler på sekunder
3. Öppet API med djupintegrationer mot kassasystem, inte bara QR-kod och digital meny
4. Inbyggd hantering av allergen- och ingrediensdata enligt EU-lagstiftning
5. En enda källa till sanning för all menydata oavsett hur många försäljningskanaler du har
- **Call-to-action:** Anslut din restaurang gratis i 30 dagar

## 🛠 Din uppgift
Bygg en MVP, minsta möjliga som bevisar värdet:
1. Få kärnflödet att funka end-to-end (det som löser problemet ovan).
2. Snyggt men enkelt UI.
3. Ett tydligt sätt att ta betalt (eller fånga intresse) enligt erbjudandet.
